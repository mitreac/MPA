# Metabolic pathway analysis for Reactome pathways

**Input:** High-throughput metabolite levels for condition and control samples. Input files: metabolite abundance table (csv, tsv, excel), sample metadata file (csv, tsv, excel), optional mapping file

**Output:** Ranked list of Reactome pathways that perturbed in the condition under study – the rank is based on a score or p-value that results from evaluating the pathway disruption given the data and the pathway information from the Reactome database. Output files: rankes pathway results(csv, tsv), network diagrams (png, jpeg), log files(TXT, HTML report)
